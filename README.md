# CCNA Study Engine

A browser-based quiz engine for CCNA 200-301 exam preparation. 910 questions across all exam domains, three study modes, and full progress tracking — no install, no account, no backend.

---

## Modes

**Section Drill** — 360 questions organised by domain. Work through VLANs, OSPF, ACLs, NAT, wireless, QoS, and more in focused sets.

**Mock Exam** — Five 50-question timed exams weighted to reflect the real CCNA blueprint distribution.

**Full Cisco-Style Exam** — Three 100-question exams simulating actual exam conditions.

---

## Coverage

| Domain | Topics |
|---|---|
| Network Fundamentals | OSI, TCP/IP, switching, cabling |
| Network Access | VLANs, EtherChannel, STP, trunking |
| IP Connectivity | OSPF, static routing, IPv6 |
| IP Services | NAT/PAT, DHCP, DNS, NTP, HSRP |
| Security | ACLs, AAA, port security, SSH |
| Automation | SDN, REST APIs, Ansible, JSON |

---

## Stack

Pure HTML, CSS, and JavaScript. Single-file build. Runs in any browser with no dependencies.
